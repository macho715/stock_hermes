# Daily Brief Prompt

Produce a read/report-only daily Hermes brief.

Required sections:
- Verdict
- Evidence
- Next action
- Safety boundary
- Repository status
- Backend checks
- Dashboard checks
- Quant dry-run status
- Open risks

Rules:
- Do not promote AMBER/RED to GREEN from advisory data.
- Do not recommend live trading.
- Cite artifact paths for every claim.
